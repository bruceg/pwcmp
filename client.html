Password Comparison Modules

Password Comparison Modules Client Library

The client library defines the following routines:

  • int pwcmp_start(const char* module) This routine sets up all necessary internal data for doing password comparisons. If module is NULL, empty, or is "plain", a built-in comparison function using strcmp is used. If module is "crypt", a built-in comparison function using crypt is used. For all other modules, an external module is started.
  • int pwcmp_check(const char* plain, const char* encod) Compare a plaintext password with an encoded one. Returns -1 if a temporary error occurred, zero if the passwords compared equal, and non-zero otherwise.
  • void pwcmp_stop(void) Shutdown the password comparison data, including closing off the comparison module.
